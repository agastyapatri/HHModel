# **Hodgkin Huxley Model of the Neuron**
Simulating the Hodgkin Huxley Neuron in Python with NumPy.

This project has two purposes:

1. SIMULATION; Simulating the Hodgkin Huxley Equations using RK4
2. DEEP LEARNING; Using the results of the simulation, train a LSTM to model the behaviour
3. EXPERIMENTATION; Compare the performance of the LSTM to the actual solution of the equations on an unseen time interval, current stimulus etc.
